# 玉龍山 延命院 ホームページ

東京都台東区元浅草にある **玉龍山 延命院** の静的ホームページです。

## 公開URL

GitHub Pages が有効になると、以下のURLで公開されます。

https://gakushuin.github.io/enmeiin/

## ファイル構成

- `index.html`：トップページ本体
- `style.css`：デザイン
- `script.js`：スマホ用メニュー
- `.github/workflows/pages.yml`：GitHub Pages公開用ワークフロー

## 公開前に確認・差し替えする情報

`index.html` 内の以下は仮情報です。正式な情報に差し替えてください。

- 電話番号：`00-0000-0000`
- メール：`info@example.com`
- 所在地：`東京都台東区元浅草`
- 参拝時間：`お問い合わせください`
- 法要・ご供養・年中行事の説明文

## GitHub Pages の設定

このリポジトリには GitHub Actions 用の公開設定を入れています。

公開されない場合は、GitHub のリポジトリ画面で以下を確認してください。

1. `Settings` を開く
2. 左メニューの `Pages` を開く
3. `Build and deployment` の `Source` を `GitHub Actions` にする
4. `Actions` タブで `Deploy static site to Pages` が成功しているか確認する

## 編集方法

文章を変える場合は `index.html`、見た目を変える場合は `style.css` を編集してください。
